# Flipkart Laptop Data Analysis

This project scrapes laptop data from Flipkart and performs data cleaning and analysis using Python.

## 🔍 Features
- Scrapes laptop data (price, rating, specs)
- Cleans and processes raw data
- Filters laptops under ₹50,000
- Identifies:
  - Top 5 cheapest laptops
  - Top 5 highest rated laptops

## 🛠️ Tech Stack
- Python
- BeautifulSoup
- pandas

## 📊 Output
- Cleaned_Ecommerce_Data.csv

## 🚀 What I Learned
- Web scraping using BeautifulSoup
- Data cleaning and preprocessing
- Handling missing and messy data
- Basic data analysis using pandas
